# Hintakartta Brand Research — Competitive Analysis & Design Trends

## Executive Summary

This report analyzes the branding landscape across Finnish, Nordic, and global proptech services, plus current design trends for dark-theme data visualization products. The goal: identify positioning opportunities for Hintakartta to stand out as a distinctive, premium Finnish proptech brand.

**Key finding:** The Finnish and Nordic proptech market is dominated by listing platforms (Oikotie, Etuovi, Hemnet) with friendly, approachable branding. There is a massive white space for a **data-first, dark-theme, map-native** brand — no competitor occupies this position in Finland or the Nordics.

---

## 1. Finnish Proptech Competitors

### Oikotie (oikotie.fi)
- **Owner:** Schibsted / Vend (rebranded 2025)
- **Brand colors:** Orange/coral primary, dark navy secondary
- **Logo:** Wordmark with arrow motif ("oikotie" = shortcut)
- **Typography:** Clean sans-serif, modern
- **Tone:** Friendly, everyday, approachable — "your shortcut to finding things"
- **Visual style:** Bright, photo-heavy, consumer-friendly
- **Key insight:** Oikotie is a horizontal marketplace (jobs, cars, apartments) — not data-focused. Brand is generic marketplace, not specialist.
- **Brand guideline:** Published on [Figma](https://www.figma.com/proto/WP0PsI3UlM6TIFKfNne4G5/Oikotie-Brand-Guideline)
- **Redesign:** Codemate unified the UX across multiple apps into a "super app"

### Etuovi.com
- **Owner:** Alma Media
- **Brand colors:** Blue primary, white secondary
- **Logo:** Simple wordmark "etuovi.com"
- **Tone:** Trustworthy, traditional — "#1 housing site in Finland" positioning
- **Visual style:** Clean, photo-centric listings, conventional real estate portal
- **Key insight:** Highest brand recall among Finnish real estate services, but visually conservative. No data visualization, no map innovation. Pure listing portal.

### Asuntojen hintatiedot (asuntojen.hintatiedot.fi)
- **Brand:** Government/Tilastokeskus aesthetic — functional, no branding effort
- **Visual style:** Utilitarian data tables, minimal UI
- **Key insight:** The closest concept to Hintakartta (actual price data), but zero brand/UX effort. This is the gap Hintakartta fills — same data promise, beautiful execution.

### Gap Analysis — Finland
| Attribute | Oikotie | Etuovi | Hintatiedot | **Hintakartta opportunity** |
|---|---|---|---|---|
| Data visualization | None | None | Basic tables | **Advanced map viz** |
| Map experience | Basic search map | Basic search map | None | **Core product** |
| Dark theme | No | No | No | **Differentiator** |
| Brand sophistication | Medium | Medium-low | None | **Premium/expert** |
| Price estimates | No | No | Historical only | **Predictive + per-building** |

## 2. Nordic Proptech Competitors

### Hemnet (hemnet.se) — Sweden
- **Brand colors:** Signature muted greens (green-on-green palette), deliberately minimal
- **Logo:** Simple wordmark with leaf/house mark
- **Typography:** Clean sans-serif
- **Tone:** Calm, trustworthy, Swedish — "the place where you find your home"
- **Visual style:** Deliberately minimal on property pages — "people come for the pictures first." Visual expression is conservatively green to avoid distracting from listings.
- **Design system:** Comprehensive, built from the ground up. Went from "immature and incoherent" to "elaborate, cohesive and appropriate"
- **Key insight:** Hemnet is the gold standard for Nordic proptech branding. But it's a listing platform, not a data tool. The green palette is distinctive but wouldn't work for price data visualization.

### Booli (booli.se) — Sweden
- **Brand colors:** Blue primary
- **Logo:** Simple wordmark
- **Tone:** Data-smart, analytical
- **Visual style:** Clean interface focused on property valuations and market data
- **Key insight:** Closest Nordic competitor to Hintakartta conceptually (property data + valuations). But light-theme, conventional layout. No distinctive map visualization. Shows that the data-analytics proptech niche exists in the Nordics — but nobody has done it with visual excellence yet.

### Boliga (boliga.dk) — Denmark
- **Focus:** Property price data and transaction history
- **Visual style:** Traditional, data tables, light theme
- **Key insight:** Another data-focused proptech. Functional but unremarkable visually.

### Finn.no — Norway
- **Brand colors:** Blue primary (classic marketplace blue)
- **Logo:** Friendly wordmark
- **Tone:** Broad marketplace (like Oikotie)
- **Key insight:** Parent company (Schibsted/Vend) same as Oikotie. Generic marketplace brand.

### Nordic Design Patterns
- **Dominant colors:** Green (Hemnet), Blue (Booli, Finn.no, Etuovi), Orange (Oikotie)
- **Missing:** Dark themes, bold accent colors, data-native aesthetics
- **Typography:** Universally clean sans-serif, nothing distinctive
- **Pattern:** All Nordic proptech brands optimize for friendliness and approachability over expertise and sophistication
- **White space for Hintakartta:** Expert, data-driven, dark-theme, visually distinctive

## 3. Global Map-Based Proptech

### Zillow (zillow.com) — USA
- **Brand colors:** Primary blue `#006AFF`, deep navy `#001751`, gold accents `#FFD237`, black `#111116`
- **Logo:** Blue wordmark "zillow" with roofline above the "ill"
- **Map style:** Light map with purple/blue pins. Dull map tones make pins stand out.
- **Data presentation:** "Zestimate" prominently displayed on every listing. Price confidence ranges.
- **Tone:** Authoritative, data-confident — "the most-visited real estate website"
- **Key insight:** Zillow proved that a **price estimate brand** can become dominant. The "Zestimate" is a household word. But their map is conventional — light tiles, colored pins. No dark theme, no heatmap-style visualization.

### Redfin (redfin.com) — USA
- **Brand colors:** Red `#A02021` primary (deliberately chose red in a blue-dominated industry), greens for "For Sale," blues for "Sold," orange for "Hot"
- **Logo:** Red house+pin icon — immediately communicates property + location
- **Map style:** Clean, functional. Color-coded pins by status. Accessibility-optimized for color vision deficiencies.
- **Tone:** Consumer advocate, "putting the customer first" (Caregiver archetype)
- **Key insight:** Redfin's map is the primary interface — map-first navigation. Their color system for map pins (green=sale, blue=sold, orange=hot) is practical but not visually distinctive as a brand.

### Rightmove (rightmove.co.uk) — UK
- **Brand colors:** Turquoise + brighter blue + cherry red (heart-shaped house icon). Moved away from conservative blue/green.
- **Logo:** Green house icon that doubles as a heart shape
- **Tagline:** "Find Your Happy"
- **Tone:** Friendly, emotional, humanized
- **Key insight:** Rightmove deliberately moved AWAY from data/expert positioning toward emotional/friendly. Their rebrand by The Team agency added "expressive doodles" and playful elements. Opposite direction from what Hintakartta should do.

### Zoopla (zoopla.co.uk) — UK
- **Brand colors:** Signature purple (deepened and refreshed in 2021 rebrand)
- **Logo:** Purple wordmark with "journey line" ribbon device
- **Design system:** "Journey lines" concept — a ribbon weaving through typography, illustrations, and infographics symbolizing that "moving home is never as simple as A to B"
- **Tone:** Sophisticated, design-forward among UK proptech
- **Key insight:** Zoopla is the most design-conscious UK proptech brand. Their "journey lines" concept shows how a single graphic device can create a cohesive identity. Worth studying for how to create a distinctive visual language.

### Map-as-Brand Observations
- **No proptech brand uses its map visualization as the brand itself** — maps are functional tools, not identity elements
- **Kepler.gl** (Uber's open-source geo tool) comes closest to "map-as-brand" — its dark-theme, glowing data visualization style is immediately recognizable
- **Opportunity:** Hintakartta's dark map with smooth price gradients could become the brand itself — the distinctive visual that people associate with the service

## 4. Design Trends for Data Products (2025-2026)

### Color Palettes — Dark Theme
- **Dark mode is now expected**, and the best sites treat it as an intentional brand expression, not just a toggle
- **"Mood mode"** — designed palettes with intentional dark tokens, not just inverted colors
- **Frosted glass** with colorful tinting (not white frost) is trending
- **Mesh gradients** with multiple color directions for depth
- **Neon micro-accents** work well in SaaS/tech-forward UI — small pops of bright color against dark backgrounds
- **Deep brown as the new black** — offers high contrast with warmth
- **Accessibility critical:** Avoid "grey-on-grey fatigue." CTAs must remain visible.

### Color for Data Visualization
- **Sequential palettes** (single hue, varying intensity) work best for price data — immediately intuitive (light=low, dark=high or cool=low, warm=high)
- **Kepler.gl aesthetic:** Dark background + glowing data overlays is the gold standard for geo-data visualization
- **Avoid:** Rainbow/spectral palettes (poor for colorblind users), harsh gradients

### Typography for Data Products
- **Recommended fonts:**
  - **Inter** — handles data well, distinctive l/1 forms, excellent for UI
  - **IBM Plex Sans** — designed for data accuracy interfaces
  - **TT Norms Pro** — proportional + tabular figures, perfect for pricing
  - **Roboto, Source Sans Pro, Noto Sans** — reliable alternatives
- **Numbers:** Use **tabular numerals** (not monospace) for price alignment
- **Rule of 2-3:** One font for data, another for supporting text
- **Trend:** Variable fonts gaining adoption for responsive scaling

### Logo Design Trends (2025-2026)
- **All-lowercase wordmarks** feel human and helpful
- **All-uppercase** projects confidence and clarity (better for B2B/fintech)
- **Spurless geometric sans-serifs** — sleek, geometric, approachable
- **Neo-minimalism** (Minimalism 3.0) — clarity with added nuance and warmth
- **Logo systems** — adaptable mark + icon + monogram + wordmark for different contexts
- **Royal blue** emerging as dominant trust color
- **Bold black + white** with confident negative space and precise geometry
- **Dated:** Overly complex marks, generic gradients, stock-icon-like logos

### Dark-First Brand References
- **Linear** — ultra-clean dark UI, monochromatic with purple accents, Inter font
- **Vercel/Geist** — systematic dark design language, Geist font family, precise spacing
- **Figma** — dark editor with vibrant tool colors
- **Arc browser** — gradient accents against dark chrome
- **Pattern:** These brands use dark backgrounds as a canvas for intentional color moments, not as an absence of design

### Map-as-Brand Opportunity
- **Kepler.gl** — dark map + glowing hex bins/arcs = immediately recognizable aesthetic
- **No proptech brand has claimed this territory** — Hintakartta could be first
- Hintakartta's Voronoi tessellation with IDW-interpolated smooth gradients on a dark basemap is already visually distinctive — this should BE the brand

## 5. Strategic Positioning Recommendations

### Where Hintakartta Should Position

```
                    EMOTIONAL / FRIENDLY
                          |
        Rightmove    Oikotie    Hemnet
                          |
   LISTING -------- + -------- DATA/ANALYTICS
   PLATFORM         |         TOOL
                          |
        Etuovi       Zoopla    Booli
                          |
                   EXPERT / SOPHISTICATED
                          |
                    *** HINTAKARTTA ***
```

### Positioning Statement (draft)
**"Suomen asuntohinnat, nähtynä."** (Finland's housing prices, visualized.)

Hintakartta occupies the intersection of:
1. **Data expertise** (like Booli/Zillow) — not a listing platform
2. **Visual sophistication** (like Linear/Kepler.gl) — dark theme, premium aesthetic
3. **Finnish specificity** — built on Finnish open data (Tilastokeskus, Ryhti)
4. **Transparency mission** (like Glassdoor for salaries) — democratizing price data

### Brand Archetype
**The Sage** — knowledgeable, truth-seeking, data-driven. Not the friendly marketplace (Oikotie) or the emotional journey (Rightmove), but the expert tool that reveals hidden truths about housing prices.

### Key Differentiators to Express in Brand
1. **Dark map is the product** — the distinctive visualization IS the brand
2. **Per-building precision** — not just area averages, every building gets an estimate
3. **Open data transparency** — built on public data, algorithm is explainable
4. **Nordic design quality** — Scandinavian minimalism meets data visualization excellence

## 6. Competitive Color Landscape

| Brand | Primary | Secondary | Theme |
|---|---|---|---|
| Oikotie | Orange/coral | Navy | Light |
| Etuovi | Blue | White | Light |
| Hemnet | Muted green | Green variants | Light |
| Zillow | Blue `#006AFF` | Navy + Gold | Light |
| Redfin | Red `#A02021` | Green/Blue pins | Light |
| Rightmove | Turquoise + Red | Blue | Light |
| Zoopla | Purple | Journey-line graphics | Light |
| Booli | Blue | — | Light |
| **Hintakartta** | **TBD** | **TBD** | **DARK** |

**Every single competitor uses a light theme.** Hintakartta's dark theme is not just a preference — it's a fundamental brand differentiator.

### Color Directions to Explore (Phase 2)
- **Option A: Warm gradient** — Amber/gold to deep red (heat-map aesthetic, "price = temperature")
- **Option B: Cool precision** — Cyan/teal accent on dark (Linear/Vercel aesthetic, "data = clarity")
- **Option C: Nordic aurora** — Purple to teal gradient (distinctive, references northern lights)

### Recommended Data Visualization Palette
For price-per-sqm (continuous sequential data), a custom ramp based on Viridis principles:
- **Low prices:** Deep indigo `#1A1147` (blends with dark basemap)
- **Mid prices:** Teal `#0D9488`
- **High prices:** Warm amber/gold `#F59E0B`
- Avoid red-green combinations (colorblind safety)
- On dark backgrounds, ramp should go from near-background dark to vivid highlight — never use pure white as the high end
- Reference palettes: Kepler.gl "Sunrise" (purple->orange), "Global Warming" (green->yellow->red)

## 7. Detailed Design Recommendations

### Typography Stack
| Role | Font | Notes |
|---|---|---|
| **UI / Body** | Geist Sans or Inter | Variable font, Finnish diacritics support, aligns with Next.js/Vercel stack |
| **Prices / Numbers** | Same font, `tabular-nums` | Essential for column alignment in price lists |
| **Logotype** | Satoshi or custom geometric sans | Distinctive character while remaining legible |
| **Code/Technical** | Geist Mono or JetBrains Mono | For any data readout overlays or tooltips |
| **Hero numbers** | 32-48px bold weight | Make price data the visual hero — large, confident display |

### Dark Theme Principles (from Linear, Vercel, Arc analysis)
1. **Background is a brand decision:** Pure black `#000` (Vercel) vs. near-black `#0D0D0D` vs. dark navy `#0A0E1A` (recommended for Hintakartta — cool tone aligns with CartoCDN Dark Matter)
2. **Accent color matters 10x more in dark UI** — it's the first thing people notice. One primary warm accent (amber/gold) used sparingly.
3. **Text hierarchy via opacity:** Primary white, secondary 70%, tertiary 50%, disabled 30%
4. **Borders:** Semi-transparent whites `rgba(255,255,255,0.06-0.12)` not solid gray lines
5. **Glassmorphism/blur** for floating panels over the map — perfect for sidebar and controls
6. **Subtle background noise/gradients** to avoid "flat black" feeling (Linear does this)
7. **Marketing consistency:** Website, social media, all materials dark. Don't have a bright white marketing site for a dark product.

### Logo Direction
- **Avoid:** House + map pin (every proptech does this), generic globe/pin icons
- **Explore:** Abstract mark derived from **Voronoi tessellation pattern** — highly distinctive, directly tied to product identity. A cluster of Voronoi cells as a logo mark would be unique in the industry.
- **Alternative:** Topographic contour lines suggesting "terrain of prices"
- **Wordmark:** "Hintakartta" in geometric sans-serif, potentially all-lowercase for approachability or all-uppercase for authority
- **Logo system:** Adaptable glyph (works at 16x16 favicon) + full wordmark + combined mark for different contexts
- **Color:** Brand accent (amber/gold) connects logo directly to the high-value end of the map palette

### Map-as-Brand Strategy
The Voronoi visualization with smooth IDW-interpolated price gradients on a dark basemap is **already visually distinctive and unlike any competitor.** This should be the brand identity itself:

1. **Signature crop:** Use a recognizable Helsinki area (e.g., Eira/Kaivopuisto peninsula showing dramatic waterfront premium) as the hero image across all materials
2. **Animated map hero:** Slowly panning/zooming live map on landing page (Kepler.gl and Mapbox approach)
3. **Consistent color story:** Same palette from map appears in all brand materials — website gradients, presentation backgrounds, social media templates
4. **Social media format:** Map crops with price annotations are inherently shareable ("Tiesitkö, että Kallion ja Eiran neliöhintaero on X euroa?"). The distinctive dark-map style makes Hintakartta content recognizable in feeds.
5. **Reference brands that achieved this:** Strava (orange heatmap = brand), Windy.com (weather viz = brand), Flightradar24 (yellow planes on dark map = brand)

### Brand Positioning Framework
| Element | Recommendation |
|---|---|
| **Archetype** | The Sage — knowledgeable, truth-seeking, data-driven |
| **Tagline options** | "Suomen asuntohinnat, nakytyna." / "Jokaisen suomalaisen oikeus ymmartaa asuntomarkkinaa" |
| **Trust signals** | Open data (Tilastokeskus, OSM, Ryhti), algorithmic transparency, 318K buildings |
| **Differentiation** | Not a listing platform — reveals the underlying market reality |
| **Tone** | Expert but accessible. Data-confident, not salesy. Finnish directness. |
| **Mission** | Democratize housing market intelligence — price transparency as a public good |
